Abstract

Computer viruses come in different forms and each type spreads in its own way. One common type of virus is a file infector virus which attaches itself to files on a computer and when someone opens up those files the virus spreads. Another type of virus is the macro virus, which is specifically found in Word or Excel files. Like other viruses, these also activate when the file is opened. Boot sector viruses infect the part of the computer that starts up the system, which makes this specific type of virus dangerous. On the other hand, polymorphic viruses can change their code to avoid being detected by antivirus software, so it makes them harder to identify and remove.

Abstract

There are different types of computer viruses, and they vary in how they work and spread across computers. One type is a resident virus, which hides in the memory of a computer and affects files without the owner noticing. Direct action viruses activate instantly when a file is opened, which makes them easier to detect. Another type of virus is a browser hijacker, which changes browser settings and redirects users to unsafe websites. Multipartite viruses are more advanced because they can infect multiple parts of a computer system at the same time. All these examples show how different and complicated computer viruses can be.
